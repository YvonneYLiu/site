# 语联丝路（静态 HTML 网站包）

## 使用方式
1. 直接双击 `index.html` 即可打开。
2. 若需更稳定的本地体验，可使用任意静态服务器（如 VS Code Live Server）。

## 目录说明
- `index.html`：主页面（首页 + 目的地 + 线路 + 景点库 + AI + 服务 + 关于）
- `assets/styles.css`：视觉与响应式样式
- `assets/app.js`：交互逻辑（三语切换、筛选、地图联动、AI模拟、一致性校验）
- `data/content.js`：结构化内容库（景点/线路/三语文案）
- `data/网站内容-初稿.pdf`：原始附件副本
- `data/pdf-import-raw.txt`：从 PDF 解码后的可检索文本
- `data/import-manifest.json`：导入元数据

## 已实现要点
- 中/越/韩三语切换与字段回退提示
- 景点结构化内容模型（ID、国家/城市、核心展品、丝路关联、政策支持）
- 路线推荐、路线详情、轻量行程单导出
- AI 助手四模式（问答/讲解/规划/术语）
- 服务接口占位（票务/交通/酒店/停车）
- 三语一致性校验看板
